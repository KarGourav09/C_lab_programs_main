# C Lab Programs

A comprehensive collection of C programming lab assignments and projects, organized by weeks of coursework.

## 📂 Repository Structure

### Weekly Assignments
- **Week_02** - Basic C fundamentals
- **Week_03** - Control structures and loops
- **Week_04_&_05** - Functions and modular programming
- **Week_06_&_07** - Arrays and data structures
- **Week_08** - Pointers and memory management
- **Week_09** - Advanced pointers and structures
- **Week_10** - More advanced concepts
- **Week_11** - Final topics and integration

### Special Directories
- **C_FILE_HANDLING** - File I/O operations and file handling techniques
- **My_Projects** - Personal C projects and implementations
- **.vscode** - VS Code configuration files

## 🎯 Purpose

This repository contains:
- Lab assignments from an academic C programming course
- Hands-on exercises covering fundamental to intermediate C concepts
- Practice problems and solutions
- Personal projects demonstrating C programming skills

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/KarGourav09/C_lab_programs_main.git
   ```

2. Navigate to a specific week or project:
   ```bash
   cd Week_02
   ```

3. Compile and run a C program:
   ```bash
   gcc program.c -o program
   ./program
   ```

## 📝 Notes

- Each weekly directory contains assignments relevant to that week's topics
- File handling techniques are centralized in the `C_FILE_HANDLING` directory
